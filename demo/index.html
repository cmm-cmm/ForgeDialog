Core dialogs

The three high-level helpers, plus prompt input variants.

Animation presets

options.animation: fade / scale / slide / bounce / blur / none.

Forms

form() — a versatile sub-form dialog with a declarative field set (text, textarea, select, checkbox, radio, number, date), per-field and whole-form validation.

Wizard (multi-step dialog)

wizard(steps, options) — an animated multi-step sub-form with a progress stepper, per-step validation, Back/Next/Finish navigation, and values merged across every step into a single typed result.

Draggable dialogs

options.draggable — grab the header and move the dialog anywhere on screen, like a desktop window. Position is clamped to the viewport.

Buttons & roles

Custom button sets via open(), including the danger role.

Behavior options

closable, closeOnEscape, closeOnOverlayClick.

Dynamic dialog (update())

Opens a "processing" dialog, then swaps title/message/buttons in place after a delay.

Stacking

Multiple dialogs open at once, layered with focus restore on close.

Labels & i18n

setLabels() (global) vs. per-call options.labels.
